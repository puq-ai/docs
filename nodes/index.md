# Nodes

Nodes are the building blocks of workflows in puq.ai.  
Each node performs a specific action, handles data, or controls how a workflow behaves during execution.

This section provides an overview of all available node categories and helps you understand **what types of nodes exist** and **when to use them**.

---

## Node Categories

puq.ai groups nodes into categories based on their purpose, such as:

- Trigger Nodes  
- Core Nodes  
- Flow Control Nodes  
- AI Nodes  
- Communication Nodes  
- Productivity Nodes  
- Analytics Nodes  
- Developer Tools  
- File & Content Nodes  
- Commerce & Payments Nodes  
- Sales & CRM Nodes  
- Marketing Nodes  
- Support Nodes  
- Forms & Surveys Nodes  
- HR Nodes  
- Utilities  

Each category contains nodes designed to solve a specific class of problems.
