# Universal AI

The **Universal AI** category includes nodes that interact with modern Large Language Models (LLMs).  
These nodes allow your workflows to generate text, analyze content, create structured outputs, handle conversations, and solve reasoning tasks â€” all powered by the latest AI systems.

At the moment, this category includes two major providers:

- **Claude** (Anthropic)
- **OpenAI** (GPT models, Assistants API)

Universal AI provides a streamlined interface over these providers, so you can integrate AI into your workflows with minimal configuration.

---

# Why Use Universal AI?

Universal AI nodes are ideal when your workflow needs:

- Natural language understanding  
- Text generation or rewriting  
- Summaries, explanations, or classifications  
- Conversational responses  
- Code generation or transformation  
- JSON-structured output based on prompts  
- Token counting or model metadata  
- Message batching or long-running reasoning tasks  

These nodes are designed to make AI a simple, interchangeable component of your automation logic.
