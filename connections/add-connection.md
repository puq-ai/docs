# Add a Connection

Connections in **puq.ai** allow you to securely link external applications, APIs, and services to your workflows.

## Creating a New Connection

To add a new connection:

1. Open the left sidebar and navigate to **Connections**.
2. Click the **New Connection** button.
3. Select the service or integration you want to connect.
4. Follow the on-screen steps to provide the required credentials or authentication details.

Each connection type may require different information, such as:
- API keys or tokens  
- OAuth authorization  
- Username and password  
- Custom headers or certificates  

## Next Steps

Detailed, step-by-step instructions for each supported connection type are provided in their respective documentation pages.  
Follow those guides to complete the setup and ensure your workflows can securely communicate with external systems.
