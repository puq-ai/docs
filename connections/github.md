# GitHub
Learn how to connect GitHub to puq.ai.

## Step 1
Sign in to your GitHub account.

## Step 2
Click your profile picture in the top-right corner and select Settings from the dropdown menu.

## Step 3
From the left sidebar, click Developer settings.

## Step 4
Click OAuth Apps from the left menu.

## Step 5
Click New OAuth App.

## Step 6
Fill in the application form.

![Github]({{ '/assets/images/connections/github/000001.png' | relative_url }}){: width="60%" }

## Step 7
Click Register application.

## Step 8

The application details page will open.
Here, you can copy the Client ID and generate a new Client Secret by clicking Generate new client secret.


![Github]({{ '/assets/images/connections/github/000002.png' | relative_url }}){: width="100%" }

Use this information to complete the GitHub integration form in puq.ai.
